I am studying the mern stack for an upcoming technical assessment.

This is a fairly basic CRUD app intended to let "teachers" log in, create "bands" for their "students", and enroll those "students" in each band.

This app is not intended for production environments. Data is stored via Mongodb Atlas on the cloud.

The main purpose of this app is to touch on every key aspect of the MERN stack such that I'm prepared when I attempt the test.

I am building this app from scratch with assistance from MongoDB and express.js docs as well as ChatGPT to help plan and move quickly.
